## 🚀 Quick Start

### 1. Use This Template

Click the **"Use this template"** button above or:

```bash
git clone https://github.com/your-username/dental-clinic-template.git
cd dental-clinic-template
npm install
npm run dev
```

### 2. Customize with AI (Recommended)

Open your AI assistant (GitHub Copilot, Claude, etc.) and say:

> "Update this website for my dental clinic: [your clinic details]"

See [QUICKSTART.md](./QUICKSTART.md) for examples.

### 3. Or Edit Manually

Open `site.config.js` and update your information. That's it!

## 📖 Example Usage

### AI-Powered Update

```
User: "Hi! Update this for Smile Care Dental in Mumbai. 
       Dr. Sharma, phone +91 9876543210, 
       services: implants, whitening, orthodontics"

AI: "I'll update your clinic information..."
```

That's it! The AI handles the rest.

### Manual Update

```javascript
// site.config.js
export const siteConfig = {
  clinic: {
    name: "Your Clinic Name Here",
    founder: { name: "Dr. Your Name" },
    contact: { phone: "+1234567890" },
    // ... more settings
  }
}
```
